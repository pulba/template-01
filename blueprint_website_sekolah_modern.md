# BLUEPRINT UI/UX WEBSITE SEKOLAH MODERN
**Stack:** Astro JS, Tailwind CSS v4, Turso, Cloudflare

## FILOSOFI DESAIN
Website sekolah harus memenuhi 4 tujuan utama:
1. Meningkatkan kepercayaan publik.
2. Menyediakan informasi cepat.
3. Menjadi pusat dokumentasi kegiatan.
4. Menjadi media konversi PPDB.

**Prinsip desain:**
- Clean institutional design.
- Mobile first.
- Fast information retrieval.
- Strong CTA untuk PPDB dan kontak.
- SEO friendly content hierarchy.

## DESIGN SYSTEM GLOBAL
- **Warna:** Primary (Biru Sekolah), Secondary (Kuning/Orange), Neutral (Abu terang, putih, hitam), Success (Hijau), Danger (Merah).
- **Typography:** Heading (Bold modern sans), Body (Clean readable sans).
- **Komponen:** Button Primary/Secondary, Card Berita/Guru/Agenda, Badge, Modal, Search Bar, Mobile Drawer, Pagination, Breadcrumb.

## STRUKTUR HALAMAN (BLUEPRINT)
1. **Beranda:** Hero Slider, Quick Stats, Sambutan Kepala Sekolah, Visi Misi, Berita Terbaru, Agenda, Galeri, Direktori Guru, CTA PPDB, Footer.
2. **Profil:** Sejarah, Visi Misi, Struktur Organisasi, Fasilitas, Prestasi.
3. **Berita:** Listing (Search, Filter, Grid) & Detail.
4. **Guru & Staff:** Direktori dengan filter jabatan.
5. **Galeri:** Album Foto & Video dengan lightbox.
6. **Agenda:** Kalender & List event.
7. **Download Center:** Pusat dokumen & file.
8. **PPDB:** Jadwal, Syarat, Alur (Visual Stepper), FAQ, Daftar.
9. **Kontak:** Info, Maps, Form, WhatsApp CTA.

## GLOBAL UX FEATURES
- Announcement popup.
- Dark mode optional.
- Loading skeleton.
- Scroll to top.
- Search global instant.
- Accessibility font scaling.
